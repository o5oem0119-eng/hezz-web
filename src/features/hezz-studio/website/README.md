# HEZZ Studio Website

Updated: 2026-07-21

Status: Active HEZZ Studio website implementation

## Purpose

This directory contains the HEZZ Studio public website implementation. HEZZ Studio owns building, editing, deployment preparation, monitoring, and maintenance for its homepage, portfolio, campaign pages, Lab Notes, Guidebook and sales pages, approved checkout and delivery handoffs, disclosures, SEO, analytics instrumentation, domains, and release records.

The website role is an internal HEZZ Studio function and does not provide external client website services unless that business line is explicitly approved later.

## Canonical Documents

- `PUBLIC_WEBSITE_BRIEF.md`: current homepage experience and implementation direction.
- `PUBLIC_WEBSITE_COMPLETION_CHECKLIST.md`: ranked completion and review workflow.
- `WEBSITE_OPERATIONS.md`: ownership, handoffs, release gates, and maintenance.
- `docs/BEAUTY_CAMPAIGN_LAUNCH_PLAN.md`: cross-department campaign and product inputs.

The current draft uses approved working media and honest pre-launch CTA states while price, checkout, delivery, and contact destinations remain unfinished.

## Handoffs

HEZZ Studio advertising and model-production workflows supply approved campaign, visual, prompt, image, video, and copy inputs. HEZZ Studio Guidebook ownership supplies approved product title, offer, price, license, and sales requirements. Shared infrastructure supplies checkout, payment, order, delivery, credentials, and performance behavior.

## Release Boundary

Local implementation, preview, tests, screenshots, and draft deployment preparation are allowed. Public deployment, domain changes, analytics activation, payment activation, and publishing require explicit user approval.
